# NeuraChat – AI Conversational Assistant
NeuraChat is an AI-powered conversational assistant designed to simulate human-like interactions using Generative AI. The system processes user queries in real time and generates context-aware responses using Large Language Models (LLMs).
## Features
* Real-time chat interaction
* Context-aware response generation
* Backend API integration with AI services
* Scalable and responsive architecture
* Clean and user-friendly interface
---
## Tech Stack

**Frontend:**
HTML, CSS, JavaScript

**Backend:**
Node.js, Express.js

**AI Integration:**
Generative AI APIs (LLM-based), Prompt Engineering

**Tools:**
Git, GitHub, VS Code

---

## How It Works

1. User enters a query in the chat interface
2. Frontend sends the request to backend APIs
3. Backend processes the request and forwards it to the AI model
4. AI generates a response
5. Response is returned and displayed to the user
